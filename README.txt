AquaPOS - PHP Web POS System

Setup Instructions:
1. Copy the 'aquapos' folder to your XAMPP htdocs directory.
2. Import 'aquapos.sql' into phpMyAdmin to create the database.
3. Edit config/db.php if needed (database name, user, password).
4. Open http://localhost/aquapos/public in your browser.
